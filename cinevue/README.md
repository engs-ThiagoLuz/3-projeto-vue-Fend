# CineVue 

Um catálogo de filmes responsivo e bonito construído com **Vue.js**

---


### Pré-requisitos

Antes de começa:
* Tenha o Node.js instalado (De preferência uma versão com suporte prolongado)
* Um gerenciador de pacotes, geralmente já vem instalado.
---

###  Instalação

## Execução

* Dê download no arquivo 
* Coloque "cmd" na pasta do file explorer
* Escreva apenas "npm run dev" e está pronto o site!
